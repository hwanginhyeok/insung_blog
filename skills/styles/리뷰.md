# 블로거 [유유베어] 스타일 가이드

## 1. 말투 (어미 패턴 & 톤)

### 기본 어미
- **주요 어미**: 
  - "~갑니다잉~", "~다고하는데", "~더라구요", "~같아요"
  - "~거든요", "~거든용", "~해봐요"
  - 문장 끝에 물음표나 느낌표 다용
  
- **특징 표현**:
  - "유유베어는", "베어가", "유유베어도" - 본인 지칭을 반복적으로 사용 (캐릭터화)
  - "잉~", "용~" 같은 귀여운 어미 추가 (예: "다그런지는 모르겠지만..ㅎ")
  - "야무지게!!" - 자신만의 표현 반복

### 감탄사 & 이모지
```
"두둠칫~~~", "음.. 기대보다는", "엄청 기대하고"
🤗 🥹 😂 🫢 - 감정 표현 이모지 자주 사용
```

### 존칭 사용
- 독자를 "잇님들", "~~하시자나용", "하실 분들"로 존댓글 유지
- 하지만 전체적으로는 친근한 반말 톤 (높낮이 혼합)

---

## 2. 구조 (글 전개 방식)

### 도입부 패턴
```
1. 공시 고지 (필수): "이 포스팅은 네이버 쇼핑 커넥트 활동의 일환으로..."
2. 상황 설정: "오늘은 ~", "유유베어는 ~", "신혼때 구매한 ~"
3. 제품 소개: 제품명을 명확하게 제시
4. 구매 링크: 네이버 쇼핑 링크 삽입
```

### 본문 흐름
- **제품 정보** → **사용 방법/조리법** → **실제 사용 후기** → **장단점 평가** → **최종 평점/추천**
- 단계별로 사진과 설명을 번갈아가며 구성
- 중간중간 개인적 경험담 삽입 (예: "10군데 가까이 시켜보면서")

### 마무리 패턴
- 솔직한 평가 (긍정적이지만 약간의 아쉬움도 표현)
- 예시 평점: "3점입니다", "대만족입니다"
- 마지막에 다시 한번 제품 링크 재삽입

---

## 3. 표현 (자주 쓰는 표현 & 감성)

### 블로거 특징 표현
| 표현 | 사용 빈도 | 예시 |
|------|---------|------|
| "야무지게" | 매우 높음 | "에어프라이어로 야무지게!!", "고구마를 야무지게 돌려봅시다" |
| "거짓말 안치고" | 높음 | "진짜 거짓말 안치고 10군데 가까이 시켜보면서" |
| "~더라구요" | 높음 | "어울리지는 않더라구요", "퍽퍽하더라구요" |
| "은근" | 중간 | "은근 부담되더라구요" |
| "꼭" | 중간 | "아침을 꼭 챙겨먹고는" |

### 비유 & 생생한 표현
- "새끼 붕어들입니다" - 작은 크기를 귀여운 비유로 표현
- "고구마 유목민이 되었거든요" - 상황을 재미있게 표현
- "성인이 1~2개정도 먹으면 적당한" - 실용적 기준 제시

### 감성 키워드
- "진심" (제철에진심 - 업체명이지만 신뢰감)
- "찐맛" - 진정성 강조
- "고급 표현 피함" - 쉽고 친근한 일상 언어 사용

## 4. 길이 (문장 & 문단 구성)

### 문장 길이
- **평균 문장 길이**: 짧음~중간 길이 혼합
- 짧은 문장: "음.. 기대보다는 아쉬워서 3점입니다..ㅠ"
- 중간 문장: "집에서 간편히 먹는 붕어빵이 다그런지는 모르겠지만.. 빵부분이 바삭함은 별로 없었고"

### 문단 구성
- **한 문단의 길이**: 3~5줄 정도 (짧은 편)
- **공백 활용**: 문단 사이에 여백을
