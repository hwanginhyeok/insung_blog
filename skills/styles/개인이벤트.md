# 유유베어 블로그 글쓰기 스타일 가이드

## 1. 말투

### 어미 패턴
- **주요 어미**: "~해요", "~하더라구요", "~해야해요", "~해보려구요"
- **특징**: 부드럽고 친근한 '-아/어요' 체를 기본으로 사용
- **감탄사/의성어**: "!!", "~ㅎㅎ", "~😏", "~🥹" 이모지를 글에 자연스럽게 삽입
- **예시**: 
  - "안녕하세요 유유베어입니다!!"
  - "파워J커플 답게.. 임신도 파워 계획으로 시작해보려구요..ㅎㅎ"

### 존칭/반말 혼용
- **기본**: 존댓글 사용 (존칭 유지)
- **친근함 표현**: 반말적 캐주얼함을 존댓글 안에서 표현
  - 예: "주변에 임신을 계획중인 잇님들에게" (신조어 '잇님' 사용)
- **자기 표현**: "유유베어는 ~" 으로 주체적 표현

---

## 2. 구조

### 글의 전개 방식
1. **도입부**: 따뜻한 인사 + 개인적 상황 공유
   - "안녕하세요 유유베어입니다!!"
   - 현재 상황/배경 설명 (예: "신혼 2년차에 접어드는 유유베어도...")

2. **본론 진입 전 맥락 제시**: 왜 이 글을 쓰는지 설명
   - "주변에 임신을 계획중인 잇님들에게 도움이 되고자..."
   - 독자의 필요성을 언급하며 신뢰 구축

3. **본문**: 
   - 정보 제공 → 개인 경험담 순서로 진행
   - 실용적인 정보(링크, 파일) 삽입
   - 자신의 선택 과정과 결과 공유

4. **마무리**: 앞으로의 계획이나 추가 포스팅 예고
   - "(아직 검사결과 들으러 한번더 방문해야해서 이 병원 후기는 추후 포스팅으로..!)"

### 특징
- **단계적 공개**: 전체 시리즈 콘셉트로 다른 편 포스팅 링크 제시
- **연속성**: 같은 주제의 여러 포스팅을 시리즈화

## 3. 표현

### 자주 쓰는 표현
| 표현 | 사용 맥락 |
|------|---------|
| "내돈내산" | 광고성이 아닌 솔직한 후기임을 강조 |
| "파워J커플" | 자신의 정체성/캐릭터 표현 |
| "~더라구요" | 주변 경험이나 정보 습득 과정 |
| "이것저것" | 일상적이고 편한 표현 |
| "괜찮은가" | 객관적 평가 기준 제시 |

### 감성 키워드
- **긍정/로망**: "어바웃타임 로망", "갬성의 완성"
- **솔직함**: "내돈내산 솔직후기", "거의 대부분"
- **공감**: "지인들에게 물어보면", "주변에"

### 비유/표현
- "결혼준비 시즌2"
- "뭔가 너무 인위적이지 않으면서"
- 이모지를 마침표처럼 사용: "때문이었어요. 😗"

## 4. 길이

### 평균 문장 길이
- **짧은 문장 주로 사용**: 1~2줄 길이가 표준
- **예시**:
  - "포르투 스냅 업체들이 엄청 많은데"
  - "메건그라피를 선택했던 이유는 작가님 인스타에 있는 스냅 사진들의 구도, 분위기가 마음에 들었기 때문이었어요."

### 문단 구성
- **단락 길이**: 3~5문장으로 짧게 끊음
- **시각적 분리**: 이모지(🥹, 😏, ✅ 등), 줄바꿈으로 시각적 쉼표 제공
- **핵심 정보 강조**: 별점 체계(★★★★☆), 체크리스트(✅) 사용
- **평가표**: 표 형식으로 정보 정렬
  ```
  적합성 : ★★★☆☆
  가격대 : ★★★★★
  ```
