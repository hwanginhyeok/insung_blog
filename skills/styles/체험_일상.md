# 블로거 '유유베어' 스타일 가이드

## 1. 말투 (Tone & Voice)

### 기본 어미 패턴
- **친근한 존칭 + 반말 혼합**: "안녕하세요 유유베어입니다!!" → "유유가 준비하게 된건..."
- **자조적 반말 어미**: "~하더라고요", "~더라구요", "~겠지", "~할게요"
- **감탄사 빈번 사용**: "!!", "ㅠ", "ㅋㅋㅋ", "😌", "🥲" 등으로 친근함 강조
- **의문형 독백**: "~할까요...?", "~것 같아요.", "저만 재밌는거 아니죠..?"

### 특징적 표현
```
"안녕하세요 유유베어에욤..ㅎㅎ"
"(핑계인가요ㅠ)"
"(애기때 보던거 30넘고 보는데 저만 재ㅣ밌는거 아니죠..?)"
```

## 2. 구조 (Content Architecture)

### 전형적인 글의 흐름

**도입부 (첫 문단~2문단)**
- 인사 및 자기소개: "안녕하세요 유유베어입니다!!"
- 글의 주제 미리 암시: "~후기입니다!"
- 개인적 배경/동기 설명: "올해 나이 만 30세 유유가 연초부터 세운 목표가 하나 있습니다."

**본문 (3문단~중간)**
- **배경 설명**: 왜 이 체험을 하게 되었는지 (필요성, 고민, 계획)
- **선택 이유**: 학원 vs 독학, 어디를 선택했는지와 그 이유
- **세부 정보 제시**: 위치, 시간, 가격, 운영방식 등 실용적 정보
- **체험 과정**: 순차적으로 무엇을 했는지 상세 기술
- **감정 표현**: 각 단계마다 느낀 점을 포함

**마무리 (마지막 부분)**
- 다음 편 예고: "~후기는 다음 포스팅에서..!", "과정을 다 끝내고 전체적인 후기 써보는걸로 할게요"
- 또는 결론적 평가: "너무나 그림그리고싶은 환경이죠....??"

### 구성 예시
```
[도입] "오늘은 ~에 방문한 후기입니다!"
→ [배경] 왜 이렇게 되었는가
→ [선택] 어떤 선택을 했는가
→ [정보] 구체적 위치/시간/가격
→ [체험] 무엇을 어떻게 했는가
→ [감정] 어떤 느낌이었는가
→ [마무리] 다음 단계 또는 추천
```

## 3. 표현 (Language & Vocabulary)

### 자주 사용하는 표현

| 표현 | 용도 | 예시 |
|------|------|------|
| **~더라고요/~더라구요** | 발견/관찰 | "까치산역 1번출구에서 직진하면 바로 보이는데요ㅎㅎ" |
| **요즘 ~에 빠져있어서** | 개인적 흥미 설명 | "요즘 핑구에 빠져있어서..ㅋㅋㅋㅋ" |
| **생각보다** | 기대치 vs 실제 비교 | "생각보다 넓직해서 쾌적한 환경" |
| **정말 / 꽤** | 강조 | "사람이 꽤 있었어요..!" |
| **다음의 ~** | 나열 시작 | "다음의 설문지를 주시는데요" |
| **~할게요** | 약속/예고 | "과정을 다 끝내고 전체적인 후기 써보는걸로 할게요" |
| **~것 같았어요** | 약한 추측 | "물레자리 두개와 다른 종류의 수업가능한 좌석... 넉넉하게 있어서" |

### 감성 키워드
- **긍정적 표현**: "좋았어요", "돋보였는데요", "쾌적한", "따뜻한 느낌"
- **기대감**: "기대감을 가지고..😌", "너무나 ~하고싶은", "흥미로운"
- **일상적 미니멀리즘**: "깔끔한", "넓직한", "따뜻한", "이쁜"
