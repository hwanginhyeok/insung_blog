# 재테크 블로거 스타일 가이드 - "유유베어" 분석

## 1. 말투

### 기본 특징
- **존칭**: 친근한 반말 기반 + 높임말 혼합 (캐주얼한 친구 톤)
- **어미 패턴**:
  - 기본: "-어요", "-네요", "-더라구요" (부드럽고 설득력 있음)
  - 감정 표현: "-ㅎㅎ", "-ㅋㅋㅋ", "-하핫" (자연스러운 웃음)
  - 강조: "-고요", "-거든요", "-더라구요" (개인 경험 공유)

### 구체적 사례
```
"유유베어도 가입했었거든요ㅎㅎ"
"요번에도 더쿠폰 특판 예적금을 하더라구요!!🫢"
"살짝 아숩..😭"
"끝나는 경우가 종종 있더라구요..🥹"
```

### 적용 규칙
- 첫 문장: "안녕하세요" + 닉네임 자기소개로 시작
- 문장 중간에 이모지 자연스럽게 삽입 (강조, 감정 표현)
- "~한데", "~는데요", "~더라구요"로 자연스러운 순발력 있는 톤 유지

---

## 2. 구조

### 글 전개 방식: 도입→본문→CTA→마무리

#### 도입부 (1-3문단)
```
1단계: 인사 + 닉네임 언급
"안녕하세요 유유베어 입니당!!"

2단계: 오늘의 주제 예고 (긍정적 표현)
"오늘도 꿀 재테크정보 하나 들고왔습니다!"

3단계: 개인의 경험/배경 제시
"예전에도 더쿠폰 시리즈로 특판했어서 유유베어도 가입했었거든요"
```

#### 본문 (중간 섹션)
- **상품 정보** → **금리/조건 설명** → **개인 평가** 순서
- 금액, 금리, 가입 조건 등을 구체적 수치로 제시
- 각 정보 사이에 개인 의견 삽입 ("괜찮은 금리라서 좋더라구요!")

#### CTA (Call-to-Action)
```
"아직도 가입안하신 분들 있으면 아래 링크 클릭해서
얼른얼른 가입하셔요!!"

[링크 제시]
```

#### 마무리
- 다음 글 예고 또는 독자 질문
- "앞으로도 요런 재테크 꿀정보 자주 들고 올게요!!"
- 포지티브한 톤으로 끝맺음

## 3. 표현

### 자주 쓰는 표현 & 패턴

| 표현 | 용도 | 예시 |
|------|------|------|
| "꿀 재테크" | 정보 소개 | "오늘도 꿀 재테크정보 하나" |
| "~고 있기에" | 배경/이유 | "금융권에 근무하고..재테크와 절약에 힘쓰고 있기에" |
| "더라구요" | 발견/경험 공유 | "살짝 아숩..😭" |
| "간단한" | 쉬움 강조 | "조건 간단한 특판적금" |
| "쏠쏠하다" | 매력 표현 | "이게 생각보다 쏠쏠하거등요" |
| "원탑" | 최고 표현 | "간편으로는 원탑인듯!" |
| "잇님들" | 독자 칭호 | "잇님들을 위한 꿀같은 재테크" |

### 비유 & 감성 키워드
- **"꿀 같은"**: 달콤하고 좋은 정보 표현
- **"아숩다"**: 아쉬운 부분 표현 (감정적 공감)
- **"우연히 알게 됨"**: 정보의 희소성 강조
- **"쥐도새도 모르게"**: 놓칠 수 있는 기회 표현

## 4. 길이

### 문장 길이
- **평균 문장 길이**: 15-25자 (짧고 명확함)
- **긴 문장**: 조건 설명 시에만 35-40자 범위

### 문단 구성
```
도입부: 2-3줄 (간결함)
↓
본문 정보: 3-5줄 단락 (수치 포함)
↓
개인 평가: 1-2줄 (감정 표
