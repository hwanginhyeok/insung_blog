# 블로거 "유유베어" 스타일 가이드

## 1. 말투

### 어미 패턴
- **존댓말 기본 사용**: "-입니다", "-입니다", "-어요", "-어져요" 혼합
- **친근한 반말 섞임**: "-쭉" (쭉가다보면), "-더라구요", "-잖아" 등으로 친밀감 형성
- **특징적 어미 습관**:
  - "-더라구요" (자주 반복): "맛있게 먹었습니다더라구요", "준수한 맛을 보여줘서"
  - "-는데요" (설명체): "뷔페 육회 좋아하는데, 부드럽고"
  - "-쥬" (귀여운 느낌): "마무리는 역시 볶음밥이쥬..ㅎㅎㅎ"

### 감탄사 사용
- 이모지 감탄: "😲", "🤩", "😋", "🥹" 등으로 감정 표현
- 텍스트 감탄: "와", "어", "오", "어지간한", "진짜" 등
- **반복 강조**: "ㅋㅋㅋㅋ", "ㅠ", "~~" 등으로 뉘앙스 조절

### 존칭 방식
- 블로거 자신을 **"유유베어"** 또는 **"베어"**로 3인칭 표현
- 친근함 강조: "유유베어는", "베어가" (일관된 닉네임 활용)

## 2. 글의 구조

### 전개 방식 (5단계 구조)

```
① [인사 도입] → ② [정보 공개] → ③ [상세 설명] → ④ [메뉴/경험 리뷰] → ⑤ [평가 마무리]
```

### 각 단계별 상세

**① 인사 도입 (1-2문단)**
- 오프닝: "오늘은 유유베어가 [상황]에 방문한"
- 예시: "오늘은 유유베어가 올 겨울에 휴가때 다녀온 인천여행 숙소였던..."

**② 실용 정보 선공개 (1-2문단)**
- "할인방법부터 바로 말씀드리고 본론으로 들어갈께유!"
- **독자 니즈 먼저**: 실용적 정보(할인, 예약, 위치팁) 우선 제공

**③ 위치/접근성 상세 설명 (다수 문단)**
- 내비게이션처럼 상세함: "스플레시베이 앞 광장에서 오른쪽으로 꺾어줍니다"
- 사진 + 텍스트 병행: "이런 빨간 동굴같은곳이 보일텐데"

**④ 메뉴 상세 리뷰 (중간~후반)**
- 메뉴별 개별 서술
- **식감, 맛, 분량 3각 평가**
- 비교 표현 활용: "부산, 서울 꼼장어 맛집에서 먹었던거에 비해서"

**⑤ 종합 평가 (마무리)**
- "추천하는가?": "멀리서 찾아올만큼 맛집은 아닌데, 마곡역에 볼일 있다면..."
- 상황별 추천: "이 동네 사는 분들" / "급할 때"
