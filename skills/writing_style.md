# 블로그 글쓰기 스타일 가이드

> 이 파일은 AI가 게시물 초안을 생성할 때 참조합니다.
> 피드백을 받을 때마다 자동으로 업데이트됩니다.

---

## 말투
- 친근한 해요체 (~했어요, ~이에요, ~예요, ~네요)
- 지나치게 격식체 금지
- 가볍고 일상적인 톤

## 구조
1. **도입** (2~3문장): 방문 계기나 기대감을 자연스럽게
2. **메인** (4~6문장): 음식/장소 설명 + 사진 묘사 + 개인 경험
3. **마무리** (2~3문장): 개인 감상 + 추천 여부 + 재방문 의향

## 길이
- 본문: 500~800자
- 제목: 20~25자 (검색 유입 고려)

## 제목 패턴
- `[지역] [장소/음식] 후기 — [핵심 한마디]`
- 예: "강남 이탈리안 맛집 — 분위기랑 파스타 둘 다 완벽"

## 금지 표현
- "안녕하세요" 시작 금지
- "오늘은 ~을 소개해드릴게요" 같은 뻔한 도입 금지
- 별점 수치 나열 금지 (★★★★☆ 등)
- 과도한 광고성 표현 금지
- "맛있었어요" 단독 반복 금지 → 구체적 묘사로 대체

## 해시태그
- 총 20~25개
- 위치 태그 5개, 카테고리 태그 10개, 감성 태그 10개
