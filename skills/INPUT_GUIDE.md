# 텔레그램 입력 가이드

> 사진 보내는 법 + 메모 쓰는 법. 이것만 따라하면 됨.

---

## 보내는 순서

```
1. 사진을 순서대로 선택 (한번에 여러장 OK)
2. 마지막 사진의 캡션(설명)에 메모 작성
3. 전송
```

---

## 사진 순서

**보내는 순서 = 글에 나오는 순서.** 이 순서대로 찍어두면 편함.

```
1장: 외관/간판
2장: 내부/좌석
3장: 메뉴판 (있으면)
4~N장: 음식/제품 사진 (메뉴별 1장씩)
마지막: 분위기샷/풍경 (있으면)
```

---

## 메모 형식

### 맛집/카페/술집

```
[장소명], [역이름 또는 지역]
[누구와], [왜 감]
주소: [전체주소]          ← 선택. 있으면 글에 포함, 없으면 생략
영업: [요일] [시간]       ← 선택
전화: [번호]             ← 선택
메뉴1(한줄평)
메뉴2(한줄평)
메뉴3(한줄평)
[특이점]
[N점]
```

**실제 예시 (사실정보 포함):**

```
양옥주점, 마곡역
유유랑, 오랜만에 동네 데이트
주소: 서울특별시 강서구 마곡로 201
영업: 월~일 11:00~22:00
어향가지(새우 가득 겉바속촉 대박)
탕수육(두툼한 고기 소스 매력적)
마파두부(연두부라 부드럽고 마라향 적당)
연태토닉 세트 레몬까지 센스
중식주점 중에 원탑
5점
```

**실제 예시 (사실정보 없이 — 이래도 OK):**

```
홀리즉떡, 마곡역
혼자, 점심
떡볶이 보통맛+치즈추가(치즈 양 적어서 아쉬움)
볶음밥(미나리+떡볶이국물 조합 찐맛)
급하게 찾아들어간 치고 준수
멀리서 올 정도는 아님
3점
```

### 카페

```
[카페명], [지역]
[누구와], [왜 감]
음료1(한줄평)
음료2(한줄평)
빵/디저트(한줄평) 또는 "빵 스킵"
[분위기 한줄]
[특이점]
[N점]
```

**실제 예시:**

```
카페 대피, 김포
유유랑, 김현아 데이트 후
판단라떼(판단잎 향 오묘 나쁘지않음)
김포가토(아포가토+견과류 달달 담백)
빵 스킵 빵지순례해서ㅠ
힙한 편집샵 갬성 옷도 판매
애견동반 가능 야외좌석 있음
5점
```

### 제품 리뷰

```
[제품명], [협찬 또는 내돈내산]
[왜 구매/사용]
[조리법/사용법]
[외관/크기 특징]
[맛/품질 한줄평]
[N점]
```

**실제 예시:**

```
비비고 말차붕어빵, 협찬
겨울 간식
에어프라이어 조리
크기 많이 작음 한입
말차크림 가득한데 빵 바삭함 별로 말차+붕어빵 안어울림
3점
```

```
크린포유 먼지봉투, 내돈내산
LG 코드제로 정품 3개 2만원 비싸서 호환 구매
10개 1만원대 가성비
아직 첫번째라 모르겠지만 대만족
4점
```

### 체험/일상

```
[뭘 했는지], [장소]
[혼자/누구와]
[왜 하게 됐는지 한줄]
[과정 핵심 포인트들]
[결과]
[꿀팁 있으면]
[시리즈면: N편]
```

**실제 예시:**

```
운전면허 필기시험, 강서운전면허시험장
혼자
만30세 늦었지만 올해 목표
안내 엉망 헤맴, 컴퓨터시험인줄 몰랐음ㅋㅋ
80점 합격
운전면허플러스 앱 강추
시리즈 1편
```

### 개인이벤트 (임신/결혼/신혼)

```
[주제], [기관/병원/업체]
[현재 상황 한줄]
[핵심 정보 요약]
[본인 경험/선택 과정]
[결과/다음 단계]
```

**실제 예시:**

```
임신사전건강관리, e보건소
신혼2년차 올해 임신계획
여성13만 남성5만 지원, 병원마다 추가비용 다름
강서쪽 7~8군데 중 리뷰보고 선택
검사완료 결과 일주일 뒤
```

## 한줄평 쓰는 법

메뉴 옆 괄호 안에 느낀 그대로 적으면 됨. AI가 유유베어 말투로 바꿔줌.

### 맛/식감 키워드 (생각나는 대로)

```
식감: 탱글탱글, 묵직, 바삭, 겉바속촉, 부드러움, 쫄깃, 두툼
맛:  짭쪼롬, 달달, 매콤칼칼, 향긋, 녹진, 담백, 달짝지근
```

### 평가 키워드

```
강추: 원탑, 대박, 찐맛, 강추, 최고
괜찮음: 준수, 무난, 나쁘지않음, 괜찮음
별로: 아쉬움, 별로, 기대이하, 굳이
```

### 비교 (있으면 좋음)

```
"다른 파스타집에 비해 면이 특이"
"부산에서 먹은 것보다 낫진않음"
"이 동네에서는 제일 나음"
```

## 빠른 참고: 최소 vs 최적

### 최소 (이것만 있어도 글은 나옴)

```
장소명, 지역
```

### 보통 (괜찮은 글)

```
장소명, 지역
누구와, 계기
메뉴(한줄평)
N점
```

### 최적 (실제 블로그 수준)

```
장소명, 지역
누구와, 계기
주소: 전체주소
영업: 요일 시간
메뉴1(식감+맛 키워드)
메뉴2(식감+맛 키워드)
메뉴3(식감+맛 키워드)
특이점/하이라이트
비교 한줄
추천 대상
N점
```

## 사실 정보 접두사 요약

> 아래 접두사로 시작하는 줄은 **그대로 글에 반영**됨.
> 안 적으면 AI가 지어내지 않고 해당 블록을 생략함.

| 접두사 | 예시 | 글에 반영되는 위치 |
|--------|------|-------------------|
| `주소:` | `주소: 서울특별시 강서구 마곡로 201` | 📍위치 블록 |
| `영업:` | `영업: 월~일 11:00~22:00` | ⏲운영시간 블록 |
| `전화:` | `전화: 02-1234-5678` | 본문 내 연락처 |

**동의어도 인식됨**: `위치:` = `주소:`, `운영시간:` = `영업시간:` = `영업:`, `연락처:` = `번호:` = `전화:`
