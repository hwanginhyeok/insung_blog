# ISSUE: HTML 복사 시 사진 포함 (압축 방식)

**날짜**: 2026-04-06
**상태**: 구현 완료, 검증 대기
**영향**: 글쓰기 페이지 (/write) → HTML 복사 버튼

## 요구사항

1. "HTML 복사" 버튼 클릭 시 텍스트 + 서식 + **사진**이 함께 클립보드에 복사되어야 함
2. 네이버 스마트에디터에 붙여넣기하면 사진이 본문 내 올바른 위치에 삽입되어야 함
3. 전체 HTML 크기가 **5MB 미만**이어야 함 (네이버 붙여넣기 제한)

## 구현 방식

- 이미지를 Canvas로 리사이즈 + 압축 (가로 800px, JPEG 70%)
- 원본 3MB → 압축 후 ~150KB
- 10장이어도 base64 합계 ~2MB (5MB 이내)
- 4.5MB 초과 시 자동으로 이미지 제외 (안전장치)

## 검증 체크리스트

- [ ] 사진 1장 글: HTML 복사 → 네이버 에디터 붙여넣기 → 사진 + 텍스트 정상 표시
- [ ] 사진 5장 글: HTML 복사 → 붙여넣기 → 모든 사진 올바른 위치에 표시
- [ ] 사진 10장 글: HTML 복사 → 5MB 미만 확인 → 붙여넣기 정상
- [ ] "복사됨! (사진 포함)" 라벨 표시 확인
- [ ] 사진 용량이 커서 4.5MB 초과 시 "사진 제외 복사됨" 표시 확인
- [ ] 네이버 에디터에서 붙여넣기 후 발행까지 문제 없는지 확인
- [ ] 모바일 환경에서도 복사 동작하는지 확인
