## 5. 프록시 대안: LTE USB 동글 (조사 완료 2026-04-05)

> 상세 레포트: `docs/프로젝트/요구사항/IP-로테이션-기술조사.md`

### 핵심 결론

**LTE USB 동글로 프록시 비용을 83% 절감 가능**

| 방식 | 50명 월비용 | 100명 | IP 품질 |
|------|----------|-------|---------|
| 주거용 프록시 | ₩172,500 | ₩345,000 | 좋음 |
| **LTE 동글 2개** | **₩30,000** | **₩40,000** | 매우 좋음 (진짜 모바일 IP) |

- 동글 재접속 시 통신사 CGNAT에서 새 IP 배정 (15-30초 소요)
- 모바일 IP는 차단 위험이 가장 낮음 (차단 시 일반 유저도 차단되므로)
- WSL2 mirrored 네트워킹 모드로 사용 가능

### 동글 모델로 재계산 (50명, ₩9,900)

| 항목 | 프록시 모델 | 동글 모델 | 차이 |
|------|----------|---------|------|
| 프록시/동글 비용 | ₩172,500 | ₩30,000 | **-₩142,500** |
| **순이익** | ₩31,665 (6%) | **₩174,165 (35%)** | **+₩142,500** |

### 실현 상태

- 조사 완료, 기술적으로 가능 확인
- 구현은 유저 20명+ 시점에 착수 예정
- 초기 비용: 동글 7만원 + SIM 1만원 = 8만원

## 6. 핵심 인사이트

### 50명 구간이 가장 위험
- 서버+Supabase 고정비가 붙고 + 세무기장 시작 → 마진 6%까지 하락 (프록시 모델)
- **동글 모델이면 마진 35%로 유지 가능**
- **요금제를 ₩14,900으로 올리면** 50명 마진 39%로 회복 (프록시 모델 기준)

### 프록시가 최대 변동비 → 동글로 해결
- 프록시: 전체 비용의 35-50% 차지
- 동글: 고정비 ₩30,000/월로 전환 → 유저 수 무관

### 세금을 간과하기 쉬움
- 부가세 10% + 종소세가 이익을 상당히 깎음
- 간이과세자 요건(연 8,000만 미만) 유지가 초기에 유리
