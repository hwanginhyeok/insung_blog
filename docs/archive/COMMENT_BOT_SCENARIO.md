## 🔍 문제 발생 시나리오

### 시나리오 1: 네이버 로그인 차단
```
상황: 2FA 또는 캡차 발생
처리: 
  1. 쿠키 복원 실패
  2. ID/PW 로그인 시도
  3. 3회 재시도
  4. 실패 시 RuntimeError → 종료
대응: 수동 로그인 후 save_cookies.py 실행
```

### 시나리오 2: 게시물 파싱 실패
```
상황: 네이버 UI 변경으로 셀렉터 무효
처리:
  1. 첫 번째 셀렉터 실패
  2. 두 번째 셀렉터 시도
  3. ... (폰백 체계)
  4. 모두 실패 시 "게시물 없음"으로 처리
영향: 해당 블로거 스킵, 다음 블로거 진행
```

### 시나리오 3: 댓글 입력창 없음
```
상황: 댓글 비허용 게시물 또는 UI 변경
처리:
  1. 여러 셀렉터 시도
  2. 10초 타임아웃
  3. 실패 시 "댓글 입력창 없음" 로그
  4. 다음 게시물 진행
```

### 시나리오 4: 캡차 발생
```
상황: 댓글 작성 시 캡차 화면
처리:
  1. 캡차 감지 키워드 체크
  2. "캡차 감지 — 게시물 스킵" 로그
  3. success=False 기록
  4. 연속 실패 카운트++
  5. 5회 연속 시 세션 중단
```

### 시나리오 5: AI API 장애
```
상황: Anthropic API 오류 또는 타임아웃
처리:
  1. 3회 재시도 (지수 백오프)
  2. 모두 실패 시 phrases.py 폰백
  3. 고정 문구로 댓글 작성
```
