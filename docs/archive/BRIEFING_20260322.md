## 1. 프로젝트 개요

**네이버 블로그 운영 전 과정 자동화 플랫폼**

| 항목 | 내용 |
|------|------|
| 백엔드 | Python 3.12, FastAPI, Playwright (WSL2/Ubuntu) |
| 프론트엔드 | Next.js 14, React, TailwindCSS, shadcn/ui (Vercel) |
| DB | Supabase (PostgreSQL) + SQLite (로컬 운영 데이터) |
| AI | Claude API — Vision(Sonnet) + Text(Haiku) |
| 봇 | Telegram Bot API |
| 자동화 | n8n 워크플로우 |

## 4. 데이터베이스 스키마

### SQLite (로컬, 사용자별 분리)

| 테이블 | 역할 | 핵심 컬럼 |
|--------|------|-----------|
| `comment_history` | 댓글 이력 (중복 방지) | post_url UNIQUE, comment_text, success |
| `blogger_visit_log` | 일별 방문 기록 | blog_id, visit_date |
| `run_log` | 실행 통계 | bloggers_visited, comments_written |
| `post_history` | 발행 기록 | title, body, hashtags, post_url, status |
| `retry_queue` | 실패 댓글 재시도 | post_url, fail_count, next_try_at |
| `blogger_behavior` | 자동봇 의심 추적 | blog_id, pattern_score, fast_reply_count |

### Supabase (PostgreSQL, 멀티유저)

| 테이블 | 역할 |
|--------|------|
| `users` | 계정 (OAuth — 카카오/네이버) |
| `bot_settings` | 사용자별 봇 설정 (blog_id, 승인모드, 시간대, 제한) |
| `bot_cookies` | 네이버 세션 쿠키 (웹 업로드) |
| `bot_commands` | 명령 큐 (웹/텔레그램 → worker) |
| `bot_run_log` | 실행 이력 (대시보드용) |
| `pending_comments` | 댓글 승인 대기열 |
| `generation_queue` | 콘텐츠 생성 큐 (web/telegram 소스) |
| `user_personas` | 글쓰기 페르소나 |
| `persona_items` | 페르소나별 카테고리 설정 |
| `persona_feedback` | 피드백 규칙 |
| `content_calendar` | 편집 캘린더 |
| `post_analytics` | 게시물 성과 추적 |

**마이그레이션:** 22건 전체 적용 완료

## 6. 핵심 데이터 흐름

### 6.1 댓글 봇 흐름

```
스케줄(20:30) 또는 /run 명령
  │
  ▼
orchestrator.py
  │
  ├─ 1. Supabase에서 설정 로드 (제한, 승인모드)
  ├─ 2. Playwright 브라우저 시작 → 네이버 로그인
  ├─ 3. 내 블로그 댓글 작성자 수집 (admin → 모바일 폴백)
  ├─ 4. 작성자별 루프 (최대 10명):
  │     ├─ 자동봇 탐지 (≥70점 스킵)
  │     ├─ 최근 게시물 수집 (3-5건)
  │     ├─ 이미 댓글 작성한 게시물 필터
  │     ├─ AI 댓글 배치 생성 (3건/1API)
  │     └─ auto: 즉시 작성 / manual: pending 큐
  ├─ 5. 재시도 큐 처리
  └─ 6. 통계 기록 + 텔레그램 알림
```

### 6.2 게시물 발행 흐름

```
텔레그램 사진 / 웹 업로드
  │
  ▼
사진 + 메모
  │
  ├─ 1. Vision 분석 (Sonnet) → 장소, 메뉴, 분위기 추출
  ├─ 2. 카테고리 자동 분류
  ├─ 3. 초안 생성 (Haiku + few-shot + PRODUCTION_SPEC)
  ├─ 4. 해시태그 생성 (15-25개, 3단 분류)
  ├─ 5. 사용자 승인 (텔레그램 인라인 / 웹 에디터)
  └─ 6. 스마트에디터 자동화 → 네이버 발행
       ├─ 제목 입력 (타이핑 시뮬레이션)
       ├─ 이미지 업로드 (file chooser)
       ├─ 본문 입력
       ├─ 해시태그 입력
       └─ 발행 확인 → URL 반환
```
